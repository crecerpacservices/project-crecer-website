# Project Crecer Website

## Project Portfolio Management
The project portfolio showcases flagship projects managed by the founding team. To update photos:

### 1. Structure
All project images should be placed in `assets/projects/`.

### 2. Specific Projects
- **Subic International Hotel**: Place images in `assets/projects/subic/`. Name them `subic-1.jpg`, `subic-2.jpg`, etc.
- **PULP Office**: Place images in `assets/projects/pulp/`. Name them `pulp-1.jpg`, `pulp-2.jpg`, etc.

### 3. Adding More Projects
To add a new project, follow the structure in `projects.html`:
1. Create a new `div` with the project details.
2. Use a CSS grid for the gallery.
3. Link the photos to your new project folder.

## Core Features
- Responsive Design (Mobile & Desktop)
- Interactive Map (Leaflet.js)
- Contact Form (Formspree Integration)
- Icon System (Lucide)
